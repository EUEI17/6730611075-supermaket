โครงสร้างสถาปัตยกรรม MVC ของโปรเจกต์
เพื่อให้เข้าใจง่าย โปรเจกต์นี้แบ่งหน้าที่การทำงานออกเป็น 3 ส่วนหลัก:

Model (productModel.js): จัดการการเชื่อมต่อฐานข้อมูล MySQL

View (.ejs files): ส่วนแสดงผลหน้าจอ (User Interface) ที่ผู้ใช้เห็น

Controller (productController.js): ส่วนควบคุมตรรกะการทำงาน รับค่าจาก View ไปสั่งการ Model และส่งผลลัพธ์กลับไปแสดงผล

ขั้นตอนการติดตั้ง (Setup)
1.เตรียมฐานข้อมูล (MySQL)
ก่อนอื่นคุณต้องสร้างฐานข้อมูลในเครื่อง (แนะนำให้ใช้ XAMPP หรือ MAMP)
เปิด phpMyAdmin
สร้างฐานข้อมูลชื่อ: supermarket_db

2.ตั้งค่าการเชื่อมต่อฐานข้อมูล
ตรวจสอบไฟล์ models/productModel.js และแก้ไขข้อมูลให้ตรงกับเครื่องของคุณ:
user: ส่วนใหญ่คือ root
password: รหัสผ่าน MySQL ของคุณ (ถ้าไม่มีให้ใส่เป็นค่าว่าง '')
database: ต้องชื่อ supermarket_db

วิธีการรันโปรเจกต์ (Execution)
1.ติดตั้ง Dependencies
เปิด Terminal หรือ Command Prompt ในโฟลเดอร์โปรเจกต์แล้วพิมพ์คำสั่ง: npm install

2.เริ่มต้นการทำงานของเซิร์ฟเวอร์
คุณสามารถเลือกเตรียมรันได้ 2 วิธี:
โหมดปกติ: ```bash
node app.js
โหมดพัฒนา: หาdมีการติดตั้ง nodemon ไว้ ให้ใช้: npx nodemon app.js

3.เข้าใช้งานผ่านเบราว์เซอร์
เมื่อเห็นข้อความ Server running on port: 3000 และ --- เชื่อมต่อได้แล้ว! --- บนหน้าจอ Terminal ให้เปิดเบราว์เซอร์ไปที่:

http://localhost:3000
