โครงสร้างสถาปัตยกรรม MVC ของโปรเจกต์นี้โปรเจกต์นี้ใช้การแบ่งส่วนการทำงานแบบ MVC เพื่อให้โค้ดเป็นระเบียบและจัดการง่าย:Model (productModel.js): ทำหน้าที่เชื่อมต่อกับฐานข้อมูล MySQL และจัดการข้อมูล View (index.ejs, add-product.ejs): ส่วนที่แสดงผลหน้าตาโปรแกรมให้ผู้ใช้ (Admin) ใช้งาน Controller (productController.js): ตัวกลางที่รับคำสั่งจากผู้ใช้ แล้วไปสั่งการ Model ให้ทำงานตามที่ต้องการ

ขั้นตอนการติดตั้ง (Setup)
1.การเตรียมฐานข้อมูล (MySQL)
เปิดโปรแกรมจัดการฐานข้อมูล (เช่น XAMPP - phpMyAdmin)
สร้างฐานข้อมูลใหม่ชื่อว่า supermaket (สังเกตว่าสะกดตามไฟล์ SQL ของคุณ)
นำเข้า (Import) ไฟล์ supermaket.sql เพื่อสร้างตาราง products
ตารางนี้จะเก็บข้อมูล: ชื่อสินค้า (name), หมวดหมู่ (category), ราคา (price), จำนวนสต็อก (stock), และชื่อไฟล์รูปภาพ (image)

2.ตรวจสอบการเชื่อมต่อใน Code
ไปที่ไฟล์ productModel.js และตรวจสอบค่าการเชื่อมต่อ:
host: 'localhost'
user: 'root'
password: (ใส่รหัสผ่าน MySQL ของคุณ ถ้าไม่มีให้ปล่อยว่าง '')
database: 'supermaket'

วิธีการรันโปรเจกต์ (Execution)
1.ติดตั้ง Library ที่จำเป็น
เปิด Terminal ในโฟลเดอร์ของโปรเจกต์ แล้วใช้คำสั่งเพื่อติดตั้ง dependencies ที่ระบุไว้ใน package.json:
npm install

2.เริ่มต้นระบบ (Start Server)
เมื่อติดตั้งเสร็จแล้ว ให้รันคำสั่งเพื่อเปิดใช้งานเซิร์ฟเวอร์:
node app.js
หากขึ้นข้อความว่า Server runniong on port: 3000 และ --- เชื่อมต่อได้แล้ว! --- แสดงว่าระบบพร้อมทำงานครับ

3.เข้าใช้งานผ่าน Browser
เปิด Google Chrome หรือ Browser อื่นๆ แล้วพิมพ์:
http://localhost:3000
